# Bitasmbl-Simple-Budget-&-Expenses-Dashboard-920feb-Empty

## Description
Build a web application that allows users to track their income and expenses, visualize monthly spending patterns with charts, and gain insights into their financial habits. Focus on simplicity, intuitive charts, and interactive summaries.

## Tech Stack
- SvelteKit
- Redux
- Recharts

## Requirements
- Allow users to input income and expense entries
- Categorize expenses and show monthly totals
- Visualize spending patterns using interactive charts
- Allow users to edit or delete previous entries
- Provide a summary of monthly or yearly budgets

## Installation
bash
git clone https://github.com/MrBitasmblTester/Bitasmbl-Simple-Budget-&-Expenses-Dashboard-920feb-Empty.git
cd Bitasmbl-Simple-Budget-&-Expenses-Dashboard-920feb-Empty
npm install


## Usage
bash
npm run dev


## Implementation Steps
1. Initialize SvelteKit project structure.
2. Configure Redux store for income and expense state.
3. Build forms to add, edit, and delete entries.
4. Implement expense categorization and monthly totals views.
5. Integrate Recharts to display interactive spending charts.
6. Create monthly and yearly budget summary views.
7. Connect UI components to Redux and ensure state updates reflect in charts and summaries.
